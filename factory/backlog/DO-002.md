Replace the two separate collapse/expand buttons with a single toggle button that changes icon based on state:
- When expanded (default): show collapse icon (chevrons-down-up), clicking collapses all
- When collapsed: show expand icon (chevrons-up-down), clicking expands all

## Changes to outlineWindow.ts

1. Replace the two button creation in `_createElement()` with a single button. Store it as `private _collapseToggleBtn!: HTMLDivElement;`
2. Add `private _allCollapsed = false;` state field
3. Create one `clickable-icon` div with aria-label "Collapse all", set icon to `chevrons-down-up`
4. On click, call a new `_toggleCollapseAll()` method that:
   - If `!_allCollapsed`: run existing `_collapseAll()` logic, set `_allCollapsed = true`, update icon to `chevrons-up-down`, update aria-label to "Expand all"
   - If `_allCollapsed`: run existing `_expandAll()` logic, set `_allCollapsed = false`, update icon to `chevrons-down-up`, update aria-label to "Collapse all"
5. In `update()`, when headings are rebuilt (full rebuild path, not early return), reset `_allCollapsed = false` and reset icon to `chevrons-down-up` since the heading list was recreated fresh
6. Keep `_collapseAll()` and `_expandAll()` as private helpers (no changes to their logic)

## Changes to styles.css

No changes needed - the existing toolbar styles work for a single button. The `justify-content: center` will center the single icon.
