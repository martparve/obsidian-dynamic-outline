When dragging horizontally to change heading level, display a vertical alignment ruler spanning the full modal height. The ruler is light gray and aligns with headings at the target level, helping the user visually "snap" to the correct indentation.

## Implementation

### outlineWindow.ts

1. Add `_rulerEl: HTMLDivElement` field.

2. In `_createElement`, create the ruler div and append to mainElement:
   ```typescript
   this._rulerEl = createEl("div", { cls: "dynamic-outline-ruler" });
   mainElement.appendChild(this._rulerEl);
   ```

3. In `_onDragOver` horizontal drag path (where `targetIndex === this._draggedIndex`), after computing deltaX:
   - Compute `levelDelta = Math.round(deltaX / 40)`
   - Compute `targetLevel = currentHeadingLevel + levelDelta` (clamp 1-6)
   - Find a li with class `li-heading-level-{targetLevel}` in the container
   - Get the left position of the `a` element inside that li, relative to the outline container
   - If no matching li exists, estimate from the dragged li's own position adjusted by the delta
   - Set `this._rulerEl.style.left = xPos + "px"` and show it

4. In `_clearDropIndicators`, hide the ruler: `this._rulerEl.style.display = "none"`.

### styles.css

```css
.dynamic-outline-ruler {
    display: none;
    position: absolute;
    top: 0;
    bottom: 0;
    width: 1px;
    background-color: var(--text-faint);
    opacity: 0.3;
    pointer-events: none;
    z-index: 1;
}
```
